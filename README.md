# Python Calculator

A simple command-line calculator built in Python that supports basic and advanced arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division (with division-by-zero protection)
- Modulus
- Power / Exponentiation
- Square Root

## How to Run

```bash
python calculator.py
```

## Usage

Run the script, select an operation from the menu (1–8), and follow the prompts to enter numbers. The result will be displayed instantly.

## Requirements

- Python 3.x
